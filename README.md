#MIA - Basic Python Voice Assistant
A simple personal assistant script built in Python that respond to voice command and perform basic tasks like searching the web, checking the time and opening aplications

#Features
Time-Based Greetings: Greets the user appropriately (Good Morning/Afternoon/Evening).

Voice Recognition: Listens for user commands via the microphone

wikipedia Search: Quickly search and reads summary from wikipedia

Web Browsing: Opens common website (yt, Google, Stack Overflow, GitHub, etc.)

System Intraction: Provides the now time and can open some (like VS Code)

#Prerequisites
Python 3.14 instaled on your system

#Installation
Install the required Python libraries using pip:

bash
pip install pyttsx3
pip install SpeechRecognition 
pip install wikipedia
#Setup
Clone or download this repository to your local machine


#Usage
Navigate to the project directory

Run the main script vityarthi.py:

bash
python vityarthi.py
speak your command (e.g., "What is the time" "Open YouTube" "Search Wikipedia for Python")

#Supported Commands
"Open YouTube"

"Open Google"

"Search Wikipedia for topic"

"What is the time?"

"Open Stack Overflow"

"Open GitHub"

"Open VS Code" (customize paths as needed for your OS)

#Notes
Ensure your microphone is working and accessible by Python

On the first run, voice libraries like pyttsx3 may require additional drivers or setups. Check package documentation if you get errors
